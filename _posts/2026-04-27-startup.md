1. Create using `jekyll new lovekarri`
2. Run `bundle info --path minima` to find location of theme files.
   1. Copy these files into the repo.
3. Comment out `theme: minima` from the `config.yml` file.
4. Remove line 5 from `_includes/head.html` - the **seo** tag.
5. Run `bundle update`
6. Run `bundle exec jekyll serve`.
7. Post away.
